# Web_Development
Our project helps students learn Data Structures and Algorithms in a fun and easy way. Normally, students only see code and theory, which can be hard to imagine.
So, we built a tool that shows algorithms with animations. For example, if you insert or delete in a stack or run a sorting algorithm, you can see it step by step on the screen.
Along with this, we added a virtual test portal where students can practice problems, write code, run it, and get instant feedback. Technical skills we have used is Frontend for HTML, CSS, JavaScript and Backend for Node.js
